因新手教程尚未实装，以下是操作建议：
第一次进入游戏时，选择普通难度即可
鼠标滚轮调整摄像头距离，按住鼠标滚轮可以拖动摄像头位置。

进入游戏后下方的观战席拥有初始符灵--符灵是友方的战斗单位。鼠标左键点击可以拖拽符灵。将其拖拽于绿色格位（战场）可以攻击敌人。
鼠标右键可以查看符灵详细信息。
战场上方为商店，花费$可以购买符灵。
符灵初始为1星，3个1星同名符灵可以升至2星符灵，以此类推。每次升星基础属性值翻倍。
键位操作：
D：刷新商店
F：$4购买4点经验。长按3秒后每帧触发一次。（已知BUG：游戏暂停时不能长按）
shift：快捷上场/下场
C：合成台，将1个符灵献祭为装备，将2个符灵合成新的，潜在更高阶的符灵。
B：背包，击败BOSS后背包中获得1件装备。
装备可以拖到符灵上装备，使符灵获得另一个技能。一般情况下（除非符灵被移除，否则）装备无法卸下。
W：工作室，可以提升所有符灵的属性值，在科技树上可以提升工作室项目的基础等级。
K：查看拥有的羁绊
A：查看拥有的能力。达成部分羁绊以获得能力。

采矿：将具有采矿技能的符灵放在矿脉上便可以采矿。目前有采矿技能的符灵：工程师、该隐、镭射光行者。
